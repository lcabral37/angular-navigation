What is this app about?

This is a simple page Angular app with a navigation through tabs which simulates a registration.
At this stage you can start the process.
In the next tab you can fill in some data in a form and test for certain attributes
The confirmation tab will shoe you all the fields you have filled in for a last confirmation
The last tab is a simple "All is done" and it will automatically reset the data. In a perfect world it would have sent the data to a backend server...
